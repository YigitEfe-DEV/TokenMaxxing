# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-06-20

### Added
- Context Maxxing Engine for context quality evaluation and scoring.
- Smart Repository mode for waste, build artifact, temp file, and duplicate file tracking.
- Context Simulator modeling workspace token sizes against context windows.
- Auto Optimization Pipeline orchestrator.
- Prompt Rewrite Engine featuring Conservative, Balanced, and Aggressive modes.
- Repository Memory summary tool mapping architecture structure and dependencies.
- Performance Dashboard rendering HTML, JSON, and Markdown reports.
- Benchmark framework evaluating counting speed, compression, and analysis rates.

## [0.1.0] - 2026-06-20

### Added
- Technical Architecture document.
- Fast/Accurate Hybrid tokenizer engine.
- Base CLI wrapper.
- Standard metadata documents.
